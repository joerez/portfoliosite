Simple 3D
Flipbook

A library for creating
simple 3D flipbooks

By: Joe Rezendes

You can become a part of the community by contributing to the Github Repo.
Once you've got your own book,
simply create your own styles
For each page the same way you
would style anything else.
The documentation is a tutorial on
setting up your book envirement.
To use in your own projects
all you need to do
is include the CSS and Javascript
files in your document.
These flipbooks are fully customizable.

I created this simple flipbook library because I wasn't happy with the current solutions on the market today

Setting up a flipbook for yourself is easy!
